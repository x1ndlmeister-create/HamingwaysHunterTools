KH Totem Tracker - Required Icon Files
========================================

Place TGA files with these exact names in this folder to use custom icons.
If a file is missing, the addon will use the standard WoW icon instead.

All files should be:
- TGA format (Targa)
- Uncompressed, 32-bit with alpha channel
- Square dimensions (e.g., 128x128, 256x256, 512x512)
- Round/circular design with transparent background
